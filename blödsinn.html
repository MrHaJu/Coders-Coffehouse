Section 1 content

The nav element represents a location on a page that links to other pages or to parts within a page. Common examples of navigation sections are menus, tables of contents, and indexes. A nav element is typically found near the top of every web page on a site, because this is where the user expects it to be.

Section 2 content

A web page can contain several nav elements. For example, one for site navigation and one for navigation of elements on the same page. Use of more than one nav element on the page is usually only necessary in complex ½websites with large amounts of information to structure.

Section 3 content

Note though that not all groups of links on a page need to be in a nav element - Only sections that consist of major navigation blocks are appropriate for the nav element. For example, a footer may contain a list of links that don't need to be in a nav element.

Pythonidae

Introduction

The Pythonidae, commonly known as pythons, are a family of nonvenomous snakes found in Africa, Asia, and Australia.

Behavior

Most members of this family are ambush predators, in that they typically remain motionless in a camouflaged position, and then strike suddenly at passing prey. Attacks on humans, although known to occur, are extremely rare
